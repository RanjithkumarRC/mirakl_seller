Installation Guide

Introduction:

Go to the 'RC Mirakl Connector - Seller' BigCommerce website and select the "Get This app" button. You should click on Install when a new page appears. Select the confirm option. Please wait for the RC Mirakl Connector - Seller to get installed.


Integration Steps for App:

  1. Login into your BigCommerce Admin Panel.

  2. Go to the ‘RC Mirakl Connector - Seller’ BigCommerce App page and click on Get this App button.

  3. A page will open, where you should click on Install.

  4. Click on the Confirm button

  5. Please wait for the 'RC Mirakl Connector - Seller' app to get installed. Once the app is installed, you should see the 'Mirakl and BC Configuration Form'.

  6. Get the Mirakl Credentials (I.e., Mirakl API URL and Seller API key) and enter the credentials in respective input boxes and click the Connection check button

  7. Once your connection is validated successfully, then click on the Save Credentials button.

  8. Once the credentials are saved you should see the Dashboard page
